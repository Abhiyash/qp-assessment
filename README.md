Grocery Booking App.

It contains the following endpoints
1) /admin/getallproducts
2) /admin/addnewproduct
3) /admin/deleteproduct/{productId}
4) /admin/updateproduct
5) /admin/manageinventory
6) /getavailableproducts
7) /placeorder
